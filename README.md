## How To Use This Project
- Using as template
- Add submodule

## How To Adding Sub Module

### Code
```zsh
git submodule add <link-repo-github> <folder-name>
```

### Sample
```zsh
git submodule add https://github.com/frogobox/piano-tiles-sub-java com
```
